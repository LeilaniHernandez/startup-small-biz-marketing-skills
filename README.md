# Marketing AI Skills

A collection of standalone AI skills for startups, small businesses, marketers, founders, and operators.

These skills are designed to help with common early-stage business tasks such as marketing, branding, content strategy, customer research, messaging, and growth.

## What These Skills Are

AI skills are structured instructions that help an AI model perform a specific task more reliably.

Instead of giving the AI a vague request, each skill gives it a clear role, process, output format, and quality standard.

## Why I Made This

I created this project as a professional development exercise while exploring how startups and small businesses can use AI more practically.

The goal was to create skills that are:

- Easy to use
- Clear and structured
- Useful for real startup work
- Not tied to one specific company
- Fully standalone
- Free of required reference files or extra attachments

## What's Included

This repository includes standalone Markdown skills for tasks such as:

- Brand positioning
- Marketing content planning
- Customer research
- LinkedIn post drafting
- Website copy review
- Startup messaging
- Growth and operations support

Each skill can be copied into an AI tool and used on its own.

## How to Use

1. Open one of the `.md` skill files.
2. Copy the full text.
3. Paste it into your AI tool of choice.
4. Add any relevant context for your business or project.
5. Run the skill and review the output.
