## What this is
 
Most "AI prompt" collections are either a wall of one-liners or a complex system you have to install and configure. This is the middle path: **seven self-contained marketing skills**, each a single Markdown file that gives an AI model a clear role, a step-by-step process, an output format, and a quality bar for one real task.
 
You don't wire anything together. You don't maintain a shared context file. You don't learn a CLI. You open the skill you need, paste it into whatever AI tool you already use, add your context, and get a focused result instead of a generic one.
 
**What makes these different**
 
- **Truly standalone** — no required reference files, no cross-skill dependencies, no install step. Each skill works on its own.
- **Tool-agnostic** — built to work in any AI chat (Claude, ChatGPT, Gemini) *and* in agent tools that read the [Agent Skills](https://agentskills.io) format.
- **Built for small teams** — written for founders and operators who don't have a marketing department, not for enterprise growth orgs.
- **Honest by design** — the skills prioritize clarity and evidence over hype, and they're upfront about when a tactic isn't worth it.

## The skills
 
| Skill | What it does |
| --- | --- |
| [**Customer Research**](customer-research/SKILL.md) | Turn interviews, surveys, support tickets, and reviews into voice-of-customer language, personas, and jobs-to-be-done — so your messaging is grounded in evidence, not assumption. |

## How to use
 
**The fast way (any AI chat):**
 
1. Open the folder for the skill you need and open its `SKILL.md`.
2. Copy the full file.
3. Paste it into your AI tool of choice.
4. Add your own context — your product, audience, and goal.
5. Run it and review the output.
**The agent way (Claude Code, Cursor, and other Agent-Skills-aware tools):**
 
Each skill is a standard `SKILL.md` in its own folder, so you can drop any folder straight into your agent's skills directory and the agent will pick it up.
 
```text
Example:
  You:   "Rewrite my homepage hero — here's the current copy and what we sell."
  Skill: marketing-copywriting → returns a sharper headline, subhead, and CTA,
         with reasoning for each choice.
```
 
> **Tip:** these skills are starting points, not autopilot. The output is only as good as the context you give it — paste in real details about your business and the result gets sharper fast.
